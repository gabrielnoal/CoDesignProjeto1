Estação Meteorológica

    Nome: Estação Meteorológica
    Curso: Engenharia Mecatrônica
    Periodo: Primeiro
    Design: Sim

  • O projeto envolveu software? Sim, para este projeto foram usados o Excel, o software do Arduino e o Fritzing para a confecção da PCB. O grupo usou o Excel para armazenar os dados das medições com cada sensor, gerar gráficos e para calcular as incertezas que foram incluídas no código de Arduino posteriormente para calibrar os sensores. O software do Arduino foi utilizado para escrever os códigos de cada sensor que foram posteriormente compilados no microcontrolador.
  • O projeto envolveu fabricação? Sim, para esse projeto utilizamos a fresadora do FabLab para a confecção da placa de circuito impresso (PCB) e também uma solda para soldar os componentes (resistores, sensores e LCD) na placa.
  • O projeto envolveu eletrônica? Sim, esse projeto é de modo geral um projeto de eletrônica. Utilizamos o Arduino Uno e diversos sensores analógicos e digitais, além de uma tela de LCD para mostrar em seu display as medições de cada sensor (temperatura, pressão, umidade relativa do ar e luminosidade). Dessa forma, pode-se concluir que bastante eletrônica foi empregada nesse projeto do primeiro semestre.
  • O projeto envolveu design centrado no usuário? De certo forma sim, pois poderíamos ter deixado a estação em uma protoboard, tornando-a mais difícil de ser manuseada e possível de se desconectar algum jumper, por exemplo. Assim, foi confeccionada uma placa para que fosse possível soldar todos os componentes sem o uso de jumpers e dessa forma fica mais simples para o usuário.
  • O projeto envolveu escrita técnica e científica? Sim, já que um relatório final foi cobrado. Nesse relatório precisamos colocar o passo a passo explicado de cada etapa do projeto (medição, fabricação, calibração) e para isso foi necessário o uso de uma linguagem cientifica, já que o projeto englobou conhecimentos de Estatística, Química, Eletrônica entre outros.
  • Qual foi o curso em que o projeto foi feito? Instrumentação e Medição.
  • Qual foi a data em que o projeto foi feito? Começo de Abril de 2016 até meio de Junho 2016.
  • Qual foi o período em que o projeto foi feito? O projeto foi feito ao longo do primeiro semestre da faculdade.
  • Qual foi a duração do projeto? Aproximadamente 2 meses.
  • Qual foi o papel do aluno no projeto? Nesse projeto me encarreguei de fazer as medições com os sensores que precisaram ser calibrados (LM35 e DHT22). Além disso, me encarreguei de escrever parte dos textos do relatório e ajudei na parte de fabricação com a solda e o software de confecção da placa.
